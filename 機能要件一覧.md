### 機能要件一覧

| 管理番号 | 機能                          | 説明                                                                                  | 分類                | 対応業務要件 | 備考                         |
|----------|-------------------------------|-------------------------------------------------------------------------------------|---------------------|--------------|----------------------------|
| FR-1     | QRコードによる自動チェックイン | 参加者がQRコードをスキャンして自動的にチェックインする機能                              | 外部機能             | BR-1         |                            |
| FR-2     | デジタルイベントガイドの提供   | 参加者にデジタルイベントガイドを提供し、スマートフォンで閲覧可能にする機能                | 外部機能             | BR-2         |                            |
| FR-3     | 写真撮影と自動アップロード     | 参加者がアプリ内で写真を撮影し、即時にサーバーにアップロードする機能                      | 外部機能             | BR-3         |                            |
| FR-4     | 画像解析とスコアリング         | アップロードされた写真を画像認識システムで解析し、スコアリングする機能                    | 外部機能             | BR-4         | AWS Rekognition APIを使用 |
| FR-5     | リアルタイム活動履歴管理       | 参加者の活動履歴をリアルタイムで収集し、表示する機能                                    | 外部機能             | BR-5         |                            |
| FR-6     | デジタル特典の発行             | 参加者のスコアに応じてデジタル特典を発行する機能                                        | 外部機能             | BR-6         |                            |
| FR-7     | データベース管理               | 参加者情報、チェックインデータ、写真データ、解析結果データ、活動履歴データ、特典データを管理 | 内部機能             | BR-1 - BR-6  | AWS DynamoDBを使用         |
| FR-8     | サーバーレス処理               | イベント全体の処理をサーバーレスで実行する機能                                         | 内部機能             | BR-1 - BR-6  | AWS Lambdaを使用           |
| FR-9     | リアルタイムデータ表示         | 収集されたデータをリアルタイムで表示する機能                                            | 内部機能             | BR-5         | Flaskを使用                |

### 説明
- **QRコードによる自動チェックイン:** QRコードをスキャンして、参加者を自動的にチェックインする機能。
- **デジタルイベントガイドの提供:** デジタルイベントガイドを提供し、参加者がスマートフォンで閲覧できるようにする機能。
- **写真撮影と自動アップロード:** 参加者が写真を撮影し、それをアプリから即時にアップロードする機能。
- **画像解析とスコアリング:** アップロードされた写真を画像認識システムで解析し、スコアリングを行う機能。
- **リアルタイム活動履歴管理:** 参加者の活動履歴をリアルタイムで収集し、表示する機能。
- **デジタル特典の発行:** 参加者のスコアに応じて、デジタル特典を発行する機能。
- **データベース管理:** すべての関連データを一元管理する機能。
- **サーバーレス処理:** イベント全体の処理をサーバーレスで実行する機能。
- **リアルタイムデータ表示:** 収集されたデータをリアルタイムで表示する機能。

これらの機能要件を満たすことで、こんこんプロジェクトの目的であるイベント参加者の体験向上と運営効率の向上を実現することができます。
