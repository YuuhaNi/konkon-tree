### 業務要件一覧

| 管理番号 | 業務                      | 説明                                                                            | 実施者         | システム化対象 |
|----------|---------------------------|---------------------------------------------------------------------------------|----------------|----------------|
| BR-1     | 参加者の受付               | 参加者がQRコードを使用してイベントにチェックインする                              | 参加者         | ○              |
| BR-2     | イベントガイドの配布       | デジタル形式でイベントガイドを参加者に提供                                        | LINE Bot       | ○              |
| BR-3     | ランドマークの探索         | デジタルガイドに従ってランドマークを探索                                         | 参加者         | ○              |
| BR-4     | 写真撮影とアップロード     | 参加者がアプリ内で写真を撮影し、即時にアップロード                                | 参加者         | ○              |
| BR-5     | 写真解析とスコアリング     | 画像認識システムにより自動で写真を解析し、スコアと解説をリアルタイムで表示            | LINE Bot       | ○              |
| BR-6     | 活動履歴の管理             | 参加者の活動履歴をリアルタイムで自動収集し、管理者端末で確認できる                  | システム管理者 | ○              |
| BR-7     | 終了時の特典付与           | スコアに応じたデジタル特典を発行し、混雑を回避して参加者に配布                      | LINE Bot       | ○              |
| BR-8     | 開発スケジュールの管理     | プロジェクトの開発スケジュールを定期的にレビューし、進捗報告を行う                  | プロジェクトマネージャー | -              |
| BR-9     | 開発コストの管理           | 開発コストを監視し、予算超過を防止する                                            | プロジェクトマネージャー | -              |
| BR-10    | 技術検証とバックアップ計画 | 技術的障害を防止するために、事前に技術検証を行い、バックアップ計画を準備する        | 開発チーム     | -              |
| BR-11    | ユーザーテストの実施       | 参加者や関係者に対してユーザーテストを実施し、フィードバックを収集・反映する        | 開発チーム     | -              |
| BR-12    | イベント前のキャンパス確認 | イベント前にキャンパス内の状況を確認し、必要に応じて柔軟に調整する                 | 運営スタッフ   | -              |

この業務要件一覧は、「こんこんプロジェクト」の全体的なプロセスを網羅し、どの業務がシステム化の対象となるかを明確にしています。
